# Mis Herramientas en Parrot OS 🦜
Este repositorio contiene mis primeros proyectos de programación y ciberseguridad.

## Proyectos incluidos:
1. **Organizador de Archivos**: Automatiza la limpieza de la carpeta Descargas.
2. **Escáner de Puertos**: Herramienta básica de red para auditar puertos abiertos.
3. **Generador de Contraseñas**: Crea claves seguras de 20 caracteres con criptografía básica.

*Aprendiendo programación y seguridad desde cero.*
